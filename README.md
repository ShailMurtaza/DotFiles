# README
These dotefiles are using i3wm with i3 blocks, conky, i3lock, ViFM, and dmenu.
It uses picom for rounded borders, shadows around windows and fadein/fadeout animations while opening and closing apps and while switching workspaces.
>Install Monokai Nerd Font or Change fonts in i3config
## Key Bindings
| Key Combination       |  Description  |
| ----------------------|:-------------:|
| $mod+Return           | Open Terminal |
| $mod+o                | Screen Lock   |
| $mod+p                | Power menu    |
| $mod+bracketleft      | Firefox |
| $mod+f                | Vifm |
| $mod+b                | homebank |
| $mod+Shift+Return     | Config edit menu |
| Print                 | Mate Screenshot |
| $mod+Print            | Take full screen screenshot usign mate screenshot |
| $mod+c                | vscode |
| $mod+u                | Start Filebrowser |
| $mod+m                | Mate system monitor |
| $mod+Shift+m          | Prismlauncher |
| $mod+semicolon        | lutris |
| $mod+Control+l        | Switch to next workspace |
| $mod+Control+h        | Switch to previous workspace |
| $mod+Ctrl+semicolon   | Move workspace to next output |
| $mod+n                | Refresh i3blocks net indictor |
| $mod+z                | Increase gaps |
| $mod+Shift+z          | Decrease gaps |
